1) Install mysql-server
	command: sudo apt-get install mysql-server

2)Verify mysql server working and log in
	command: sudo mysql
	
3)Once logged into the mysql-server, list the authentication method for root user
	a) Verify current authentication plugin
		mysql> use mysql;
		mysql> select * from user where User='root';
	b) Change authentication plugin and add password
		mysql> ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'thenewpassword';
	c) Verify changes
		mysql> select * from user where User='root';
	c) Apply changes
		mysql> FLUSH PRIVILEGES;

4) Install phpMYAdmin
	command: sudo apt-get install phpmyadmin
	
5)Link phpMyAdmin directory to nginx webroot
	command: sudo ln -s /usr/share/phpmyadmin /var/www/html/dbadmin
	
6)Log into your phpMYAdmin webconsole
	https://yourawsFQDN.ec2.aws.com/dbadmin
